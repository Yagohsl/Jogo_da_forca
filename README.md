# Jogo da Forca
### Este é um jogo da forca feito na linguagem C, utilizando a biblioteca string.h
#
![img1](https://github.com/user-attachments/assets/216ddb2e-301e-43ca-862f-c28c201573ad)


Primeiramente é preciso informar a(s) palavra(s) a ser(em) acertada(s).

#
![shot-2025-02-18_20-06-35](https://github.com/user-attachments/assets/ac57a398-eff0-4ea0-a490-a1dac783e0ba)

Após o jogo começar o jogador deve informar se quer adivinhar uma letra ou a palavra inteira, se acertar a letra ela aparecerá, senão perderá uma vida. Se o jogador acertar a palavra uma mensagem de vitória aparecerá.

#
![shot-2025-02-18_20-11-25](https://github.com/user-attachments/assets/95f44a3e-86c0-4c3f-a1b8-0509e3a47675)

Se o jogador chegar a zero vidas, aparecerá uma mensagem mostrando qual era a(s) palavra(s).
